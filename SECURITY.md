# Politique de sécurité

## Versions supportées

Le projet est maintenu sur la branche principale.

## Signaler une vulnérabilité

Merci de **ne pas** créer d’issue publique pour une vulnérabilité.

Veuillez signaler de façon privée :

- via les Security Advisories GitHub du dépôt, ou
- en contactant les mainteneurs avec un rapport détaillé.

Incluez :

- description de la vulnérabilité,
- impact estimé,
- étapes de reproduction,
- proposition de correctif si possible.

Nous ferons au mieux pour confirmer, corriger, puis communiquer de manière responsable.
